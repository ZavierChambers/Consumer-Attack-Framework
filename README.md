# Consumer ATT&CK-Lite

## Phase 1 -- Foundation

### Overview

Consumer ATT&CK-Lite is a simplified, consumer-focused adaptation of the
MITRE ATT&CK framework.\
The goal is to translate enterprise techniques into plain-English,
practical guidance for individuals.

------------------------------------------------------------------------

### Phase 1 Goals

-   Define the core data structure for attack entries
-   Identify initial consumer threat categories
-   Map sample threats to ATT&CK Technique IDs
-   Confirm feasibility of an offline static web implementation

------------------------------------------------------------------------

### Planned Data Model

Each entry will include:

-   Title\
-   ATT&CK ID(s)\
-   Description\
-   Risk\
-   Mitigations\
-   Detection Signs\
-   Difficulty Level

------------------------------------------------------------------------

### Technical Direction

Phase 1 will use:

-   Static HTML\
-   Embedded CSS\
-   Vanilla JavaScript\
-   Local data structure\
-   Full offline capability

------------------------------------------------------------------------

### Success Criteria

Phase 1 is complete when:

-   Data structure is finalized\
-   Initial dataset is defined\
-   ATT&CK mappings are validated\
-   Architecture direction is confirmed

------------------------------------------------------------------------

This phase establishes the foundation for future searchable, printable,
MITRE-style functionality.
